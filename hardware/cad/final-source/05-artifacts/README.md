# Artifacts – Auxiliary Mechanical Components

This directory contains auxiliary mechanical components used across different parts of the FrED Factory system.

These elements support interaction between the robot, the workstation, and the FrED device.

## Components

- **gripper**
  Mechanical elements related to the robot gripping mechanism.

- **interlock**
  Safety or positioning components used to secure or align elements within the system.

- **subassembly**
  Components used in the manual preparation station.

- **valve**
  Mechanical elements related to the filament process or cooling subsystem.
