# Workstation Tables – CAD Structure

This directory contains the CAD models of the workstation tables used in the FrED Factory system.

The tables define the physical workspace for both automated and manual operations within the station.

## robot

Contains the structure of the **robot workstation table**, where the KUKA robot operates.

- **parts/**
  Structural components used in the robot workstation table.

- **ws_robot.SLDASM**
  Top-level assembly of the robot workstation table.

## subassembly

Contains the structure of the **manual subassembly workstation**, where an operator prepares components before robotic manipulation.

- **parts/**
  Structural components used in the manual preparation station.

- **ws_manual.SLDASM**
  Top-level assembly of the manual workstation.
