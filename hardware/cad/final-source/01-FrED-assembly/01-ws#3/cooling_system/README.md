# Cooling System – CAD Subsystem

This directory contains the mechanical design of the FrED cooling subsystem developed in WS#3.

The cooling system is responsible for regulating the temperature of the recycled filament after extrusion, ensuring dimensional stability and material quality before the next stage of the process.

## Folder Structure

- **assemblies/**
  Subassemblies used to construct the full cooling system.

- **parts/**
  Individual mechanical components used in the subsystem.

- **printing-files/**
  Exported models prepared for manufacturing or 3D printing.

## Main Assembly

- **CoolingSystem.SLDASM**

This file represents the top-level assembly of the cooling subsystem and references the components located in the folders above.
