# FrED Assembly – CAD Structure

This directory contains the main CAD assemblies related to the FrED system.

The structure separates **subsystem development** from **system-level integration**.

## 01-ws#3

Contains the assemblies developed during Workshop #3.

This folder includes the mechanical design of the **cooling system** and its related components such as structural profiles and auxiliary elements.

Main elements include:

- Cooling system subsystem
- Structural profiles
- Auxiliary components (elbow parts)
- WS#3 top-level assembly (`WS#3.SLDASM`)

## 02-integration

Contains system-level assemblies where components from different workstations (WS#1, WS#2, WS#3) are combined.

These assemblies represent the **integration stage of the FrED system**, where subsystems are positioned and validated together.
