# Feeder – CAD Subsystem

This directory contains the CAD models related to the **material feeder subsystem** used in the FrED Factory system.

The feeder is responsible for guiding and positioning filament material during the recycling process and supporting its interaction with other mechanical components of the station.

---

# Folder Structure

## cooling-system

Contains the cooling subsystem associated with the feeder design.

Two versions of the design are included:

- **implemented-version**  
  Version that was manufactured and implemented in the workstation.

- **improved-version-not-tested**  
  Updated design proposal with improvements that were not manufactured or tested during the project.

---

## elbows

Contains the elbow components used in the feeder structure.

- **parts/**  
  Individual components used to build the elbow assemblies.

- **FeederCodosAssembly.SLDASM**  
  Assembly file referencing the elbow components.

---

## profiles

Contains the structural profiles used in the feeder subsystem.

- **parts/**  
  Individual profile components.

- **feeder_profiles.SLDASM**  
  Assembly file representing the structural profile configuration of the feeder.
