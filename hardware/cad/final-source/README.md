# FrED Factory – Final CAD Assemblies

This directory contains the finalized CAD models of the mechanical components and assemblies used in the FrED Factory system.

The designs include both **automated assemblies** integrated with the robotic station and **subassembly components** prepared by an operator before robotic manipulation.

These models represent the final structural configuration used to support the FrED automation workflow.

---

# Assembly Categories

- **01 – FrED Assembly:** Complete mechanical assembly of the FrED system structure. This includes the cooling system and the top structure of the WS#3 system.

- **02 – Holder:** Mechanical holders and supports used to secure the FrED device and position the assembly components handled by the robot.

- **03 – KUKA Robot:** CAD model of the industrial robot used in the station.  
  Robot manufacturer: **KUKA Roboter GmbH**, Zugspitzstraße 140, 86165 Augsburg, Germany.

- **04 – Tables:** Contains the tables used for both the **automated station (robot workspace)** and the **subassembly preparation station**.

- **05 – Artifacts:** Auxiliary mechanical parts and small components used across different areas of the system. These parts support assembly, positioning, or integration between subsystems.

- **06 – Feeder:** Mechanical components related to the robot feeding mechanism within the FrED process. These elements support the controlled movement and positioning of filament material.

---

## 07 – Full Stations

Complete workstation assemblies showing how individual components integrate into the final station layout.

These assemblies illustrate the interaction between:

- Robot workspace
- Subassembly preparation area
- Structural supports
- Processing equipment

---

# Role in the System

The models in this directory define the **final mechanical configuration of the FrED Factory workstation**.

The design reflects a hybrid manufacturing setup where:

- Operators prepare parts in subassembly areas
- The robot performs automated manipulation and assembly tasks
- Mechanical structures ensure alignment, stability, and integration between all subsystems

These CAD assemblies serve as the reference for the final station configuration used in the FrED Factory project.
