# FrED Factory – CAD Releases

This directory contains the **released versions of the FrED Factory CAD models**.

Each release represents a stable version of the mechanical design after completing a development phase or milestone.

The structure of this release follows the same organization as the main CAD source directory, including the following assemblies:

- 01 – FrED Assembly  
- 02 – Holder  
- 03 – KUKA Robot  
- 04 – Tables  
- 05 – Artifacts  
- 06 – Feeder  
- 07 – Full Stations  

These folders contain the finalized versions of the CAD components prepared for documentation, distribution, or reference.
