# FrED Factory – CAD Design Files

This directory contains the mechanical design files used in the development of the FrED Factory system.

The CAD models represent structural components, mechanical supports, and assemblies related to the FrED automation station and its subsystems.

These files document the physical design of the system and serve as the reference for manufacturing, assembly, and design iterations.

---

# Directory Structure

## final-source

This folder contains the original CAD source files used during the design process.

These files include editable models and assemblies that define the geometry of the system components. They are intended for further modification, design updates, and engineering analysis.

## releases

This folder contains finalized versions of the CAD models prepared for documentation or distribution.

Each release represents a stable version of the mechanical design after completing a development phase or milestone.

Example:

v1.0-final-structure  
First finalized structural version of the FrED Factory mechanical design.

---

# Design Role in the FrED System

The CAD models define the mechanical structure required for the operation of the FrED Factory system. These designs support:

- Integration of the automation station
- Mounting structures for mechanical and electronic components
- Structural elements required for the recycling process
- Interfaces between the robot cell and the subassembly station

The mechanical design ensures that the automation components, operator station, and processing units can operate within a stable and organized structure.
