# FrED Factory – Simulation

This directory contains simulation environments used to analyze and validate different aspects of the FrED Factory system.

Simulation is used to study the behavior of the manufacturing station, evaluate system interactions, and support the design of the automation workflow.

---

# Simulation Structure

- **plant-simulation**  
  Contains simulations related to the physical layout and behavior of the manufacturing station.  
  These models represent the interaction between equipment, robot workspace, and workstation structure within the factory environment.

- **process-simulate**  
  Contains simulations focused on the manufacturing process itself.  
  These simulations analyze the sequence of operations, material flow, and process behavior within the FrED system.

---

# Role in the Project

The simulation environments support the development of the FrED Factory system by allowing the team to:

- Analyze workstation behavior before implementation
- Study interactions between subsystems
- Validate automation and process concepts
- Support engineering decisions during the design phase
