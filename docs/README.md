# FrED Factory – Project Documentation

This directory contains the documentation related to the development of the FrED Factory system.

The materials stored here describe the planning, procedures, and visual records of the project. These documents provide context for the engineering decisions, development process, and progress of the system.

---

# Documentation Structure

## 01-reports

This section contains formal documents produced during the development of the project.

- **Kanban:**  
  Planning and task management materials used to organize the project workflow and track development progress.

- **SOP:**  
  Standard Operating Procedures describing how specific tasks or operations should be performed within the FrED Factory system.

- **FrED Factory – WS#3.pdf:**  
  Workshop report describing the system design, development process, and results of the project phase.

---

## 02-media

This section contains visual documentation of the project development and system components.

- **01-automation station:**  
  Images and media related to the automation station and robotic system.

- **02-subassembly:**  
  Documentation of the operator subassembly station where components are prepared before robotic manipulation.

- **03-product:**  
  Images and references related to the FrED system and the recycled filament output.

- **04-team:**  
  Photos and visual documentation of the team during the development of the project.

---

# Role in the Project

The documentation stored in this directory serves several purposes:

- Record the development process of the FrED Factory system
- Document procedures and engineering decisions
- Provide visual references of the system and its components
- Support the academic reporting requirements of the project

Together, these materials provide a comprehensive record of the project’s development and its engineering context.
